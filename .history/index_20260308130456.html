Week 3 – Building Web Pages with HTML

W3Schools MDN Web Docs

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web.

Every web page you visit is built on HTML. It works together with CSS for styling and JavaScript for interactivity.

This lab covers the fundamental building blocks of HTML, including headings, paragraphs, lists, tables, images, links, and forms.

HTML is the backbone of every website and is essential for any web developer to learn.

It is strongly recommended to validate your HTML using the W3C Validator.

Remember to always include an alt attribute on every image element.

The chemical formula for water is H2O, and the area of a circle is πr2.

Common HTML Elements

  • Headings (h1–h6)
  • Paragraphs (p)
  • Links (a)
  • Images (img)

Steps to Create a Web Page

  1. Create an HTML file with a .html extension
  2. Add the DOCTYPE declaration and html tags
  3. Add head and body sections
  4. Write your content inside the body
  5. Open the file in a browser

HTML Glossary

Element
A building block of an HTML page, defined by a start tag, content, and end tag.
Attribute
Extra information provided inside a tag, such as class, id, or href.
Semantic HTML
Using HTML elements that convey meaning about the content they contain.
HTML Elements Reference
Element Type Description
<h1> Block Main heading of the page
<p> Block Defines a paragraph of text
<a> Inline Creates a hyperlink to another resource
<img> Inline Embeds an image into the page
